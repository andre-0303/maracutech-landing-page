# MaracuTech Landing Page

Landing page institucional da MaracuTech, empresa de tecnologia agrícola focada no cultivo do maracujá.

## Sobre

Plataforma de gestão inteligente para produtores de maracujá, combinando tradição do campo com precisão tecnológica. A landing page apresenta os serviços de monitoramento de talhões, controle de manejos e inteligência financeira para agricultores.

## Stack

- **Framework**: Next.js 15
- **Linguagem**: TypeScript
- **Estilização**: Tailwind CSS v4
- **Ícones**: Lucide React
- **Fontes**: Google Fonts (Poppins)

## Getting Started

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000` para visualizar o projeto.
